mysqldump

Posted at 2019-03-17 mysql 

  • 导出创建表语句

    mysqldump –no-data -u 用户名 -p 数据库名 > 导出文件名

  • 导出数据

    mysqldump -t 数据库名 -u 用户 –hex-blob > 导出文件名

    –hex-blob 此命令是防止大文件blob格式的导出乱码

  • 导出整个数据库

  • 导出某表的部分数据

    相当于 select * from logaudit_log where id < 10 的数据以及表结构都被导入拉test.sql中

注意在windows下powershell不能执行mysqldump，导出数据格式会有问题。
