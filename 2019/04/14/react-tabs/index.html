react_tabs

Posted at 2019-04-14 react 

工作场景中遇到需要利用react的tabs动态加载某个路由内容的场景，在实践中未发现antd官方的demo。后利用router和tabs进行结合从而做了实现。

关于标签的add,remove方法参考官方示例即可。antd ^3.15.1

  • 将导航中的不同模块注册到router中

  • 左侧菜单栏中添加add方法

  • add方法
  • render

  • 刷新tabs

    刷新tabs暂时采用setState的方式进行重新渲染数据，但是要注意页面只会重新渲染发生改变的内容。

    • 例如后端分页时的currentPage，如果需要刷新后页码回到第一页则需要设置pagination.current为1.

  • 优化

    • 考虑将导入的类路由存入map进行自动匹配
