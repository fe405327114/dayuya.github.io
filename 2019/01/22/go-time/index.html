go_time

发布于 2019-01-22 go 

时间序列化
  • func Parse(layout, value string) (Time, error)
  • func ParseInLocation(layout, value string, loc *Location) (Time, error) (layout已带时区时可直接用Parse)
  • func Date(year int, month Month, day, hour, min, sec, nsec int, loc *Location) Time

推荐使用time.ParseInLocation,使用传参的时区解析时间。

时间反序列化
  • to string

注意: galang 中指定的特定时间格式为 "2006-01-02 15:04:05 -0700 MST"

  • to time stamp
时间段(Duartion)
时区(Location)
时间运算
Ticker类型

Ticker 类型包含一个 channel，处理每隔一段时间执行的业务(比如设置心跳时间等)

Timer类型

Timer 类型用来代表一个单独的事件，当设置的时间过期后，发送当前的时间到 channe，即Timer.C

以上两函数都可以使用 Reset, 使用 Reset 时需要确保 t.C 通道被释放时才能调用，以防止发生资源竞争的问题
